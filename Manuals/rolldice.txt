```
rolldice: Returns the result of a dice roll. You can optionally provide a number representing the number of sides of the die. If no number is specified, a 20-sided die will be used. The number specified must be a positive number.

Usage: !rolldice
       !rolldice [Number]

Examples:
!rolldice
- This will roll a 20-sided die and return the result.

!rolldice 100
- This will roll a 100-sided die and return the result.
```
